# amanXDtool

**Live Demo:** [https://amansandhu2718.github.io/AdobeXD_Clone_aiReady/](https://amansandhu2718.github.io/AdobeXD_Clone_aiReady/)

This folder contains the React editor and MCP-style stdio server.

## MCP Server

```powershell
npm install
node agent-tools/server.mjs
```

List exposed MCP tools:

```powershell
node agent-tools/scripts/list-tools.mjs
```

See [docs/TOOL_API.md](docs/TOOL_API.md) for MCP endpoint details.
