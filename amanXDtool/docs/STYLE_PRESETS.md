# Style Presets

Use these presets when the user does not provide a detailed brand style.

## Stitch Product Concept

Use for: general AI-generated app/site concepts.

- Mood: polished, complete, editable
- Layout: one full screen with clear sections
- Palette: one brand color, one accent, clean neutrals
- Type: Inter or similar geometric sans
- Radius: 12-24
- Shadow: soft and restrained
- Detail: theme consistency, reusable groups, clear hierarchy

## Dribbble Showcase

Use for: beautiful, creative, portfolio-worthy visuals.

- Mood: expressive, crafted, memorable
- Layout: strong focal point, layered cards, asymmetry when helpful
- Palette: bolder brand/accent contrast, still limited
- Type: large display headline plus crisp labels
- Radius: varied but intentional
- Shadow: visible but clean
- Detail: realistic content, badges, image cards, tasteful decorative structure

## Lovable Product App

Use for: app screens and product mockups that should feel buildable.

- Mood: shippable, practical, modern
- Layout: real navigation, real controls, real content states
- Palette: implementation-friendly token set
- Type: consistent scale with reusable styles
- Radius: 8-16
- Shadow: minimal, focused on hierarchy
- Detail: groups named by purpose, aligned controls, realistic data

## Food Delivery Premium

Use for: Zomato/Swiggy/DoorDash-like concepts.

- Mood: appetite, speed, local discovery
- Palette: warm red/coral, charcoal, warm white, green success accent
- Type: bold friendly headline, compact card metadata
- Components: location search, cuisine chips, restaurant cards, rating pills, delivery time, offer strip
- Imagery: image-filled food/restaurant cards
- Avoid: empty generic hero or only text without food imagery

## SaaS Calm

Use for: dashboards, admin tools, analytics, CRM.

- Mood: focused, trustworthy, efficient
- Palette: neutral shell, cool accent, restrained semantic colors
- Type: compact labels, readable numbers, clear headings
- Components: sidebar/top nav, metric cards, chart panels, table/list, filters
- Avoid: marketing hero layout inside tools

## Luxury Editorial

Use for: portfolio, travel, fashion, wellness, premium product.

- Mood: spacious, refined, confident
- Palette: restrained neutral plus one elegant accent
- Type: large display headline, minimal labels
- Components: full-bleed image areas, editorial cards, subtle CTAs, proof row
- Avoid: crowded cards and loud gradients

## AI Futuristic

Use for: AI tools, devtools, automation, cybersecurity.

- Mood: intelligent, precise, high-tech
- Palette: dark or crisp light neutral, electric accent, limited glow
- Type: geometric sans with optional mono labels
- Components: command input, graph/data cards, workflow blocks, status chips
- Avoid: excessive neon or unreadable contrast
