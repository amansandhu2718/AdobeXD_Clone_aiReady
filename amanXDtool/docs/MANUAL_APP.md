# Manual App

amanXDtool includes a browser editor for manual design work.

## Run

```powershell
npm install
npm run dev
```

Open:

```text
http://127.0.0.1:5173/
```

## Use

- Create and rename projects.
- Create artboards.
- Add rectangles, ellipses, lines, text, icons, and image-filled shapes.
- Drag, resize, rotate, group, ungroup, lock, hide, and reorder layers.
- Align and distribute selected layers.
- Use right-click on canvas layers for common object actions.
- Import/export `.amanxd.json`.
- Export PNG/JPEG assets.

## Build

```powershell
npm run build
```

The static app is written to:

```text
dist/
```
